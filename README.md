# TriviaVerse (Place holder)

A mixed pop-culture trivia web app

## Features (WIP)
- Create an account with a username and password
- Log in and play trivia quizzes
- Categories include anime, video games, movies/TV, music, and internet culture
- Timed questions
- Faster correct answers earn more points
- Leaderboard and score history
- Admin account can add, edit, and delete questions

## Profile Statistics

The profile page now shows player statistics based on completed quiz attempts:
- Quizzes played
- Total score
- Best score
- Average score
- Most-played category
- Best Random Mix score



## Public Player Profiles

Logged-in users can click a player's username on the leaderboard to view that player's quiz statistics and quiz history
