# TriviaVerse (Place holder)

A mixed pop-culture trivia web app

## Features (WIP)
- Create an account with a username and password
- Log in and play trivia quizzes
- Categories include anime, video games, movies/TV, music, and internet culture
- Timed questions
- Faster correct answers earn more points
- Leaderboard and score history
- Admin account can add, edit, and delete questions
